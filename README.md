## Snake Game (Java)

A proposta deste projeto foi recriar o Snake de forma simples, mas funcional, trabalhando conceitos importantes como lógica de jogo, controle de eventos, renderização gráfica e organização de código.

## Funcionalidades

- Movimentação da cobra com teclado
- Geração aleatória de comida
- Crescimento da cobra ao se alimentar
- Detecção de colisão (paredes e corpo)
- Sistema de Game Over
- Correção de input para evitar movimentos inválidos

## Tecnologias utilizadas

- Java
- Java Swing (interface gráfica)
- Programação orientada a objetos

## Como executar

```bash
javac SnakeGame.java
java SnakeGame
```

## Controles

- Setas do teclado para movimentação

## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de programação em Java, especialmente relacionados à lógica de jogos e manipulação de eventos.

Apesar de simples, ele demonstra a construção de um sistema interativo completo, com controle de estado, atualização em tempo real e organização de código.
